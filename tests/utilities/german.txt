Übersicht über BackOffice Server 2000-Produkte 
 
BackOffice Server 2000 ist die neue Version der marktführenden Serversuite
für das Betriebssystem Microsoft Windows® 2000. 
Microsoft BackOffice Server reduziert die Kosten sowie die Komplexität des Erstellens,
Bereitstellens und Verwaltens integrierter und skalierbarer Informationstechnologielösungen für Filialen,
Abteilungen und mittelständische Unternehmen.
Diese Version integriert Windows 2000 Server und die neuesten Standardversionen der Komponentenprodukte.
Sie stellt eine breite Palette an Infrastruktur- und Anwendungsdiensten zur Verfügung,
darunter Verzeichnis-, Netzwerk-, Webanwendungs-, Datenbank-,
Nachrichtenübermittlungsdienste und Dienste für die Zusammenarbeit sowie Internet-Proxy- und Firewalldienste,
Hostintegration und Windows-Desktopverwaltung.
